# WebCodecs API Documentation Site

A comprehensive, single-file documentation website for the WebCodecs API.

## File
- `index.html` — The entire documentation site (self-contained, no build step needed)

## Free Hosting Options

### Option 1: GitHub Pages (Recommended)
1. Create a new GitHub repository (e.g. `webcodecs-docs`)
2. Upload `index.html` to the repo
3. Go to Settings → Pages
4. Set source to `main` branch, root `/`
5. Your site will be live at `https://yourusername.github.io/webcodecs-docs/`

### Option 2: Netlify Drop (Fastest)
1. Go to https://app.netlify.com/drop
2. Drag and drop the `index.html` file
3. Instantly get a live URL

### Option 3: Vercel
1. Install Vercel CLI: npm i -g vercel
2. Run `vercel` in the folder containing index.html

### Option 4: Surge.sh
1. npm install -g surge
2. cd into folder, run: surge
